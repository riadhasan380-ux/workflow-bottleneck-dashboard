# Workflow Bottleneck Analysis Dashboard

## Overview
This project analyzes service workflow performance using Power BI to identify bottlenecks and inefficiencies.

## Key Insights
- Prep stage is the main bottleneck (~6 minutes)
- Peak hours increase total service time by ~51%
- Error rates vary across staff members

## Tools Used
- Power BI
